UX Design Brief

Project Overview

Popcornflix is a website and legal streaming service that offers free full-length movies and television shows. Users can view content using one of several supported platforms, including gaming consoles, streaming devices and web browsers.

For this project, I analyzed the performance of a website based on the usability objectives and goals of the website, then selected appropriate evaluation methods within an iterative design process to improve user interface design and user experience. Then I produced a formal report on complete website analysis, including visual documentation such as user journeys, sitemaps, and high fidelity wireframes.
